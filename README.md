# The Skincare Studio Website

A modern React website for The Skincare Studio Medical Spa in Stratford, Connecticut.

## Features

- **Modern design** – Clean layout with white, blue, and light green color palette
- **SEO optimized** – Meta tags, Open Graph, Twitter cards, and JSON-LD structured data
- **Responsive** – Mobile-friendly with Tailwind CSS
- **Pages** – Home, Services, Contact with routing

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Project Structure

- `src/pages/` – HomePage, ServicesPage, ContactPage
- `src/components/` – Header, Footer, SectionEyebrow
- `src/data.js` – Content, services, contact details
- `public/logo.png` – Company logo
